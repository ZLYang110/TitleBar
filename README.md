自定义标题栏布局，沉浸式支持，标题栏图片背景，可自定义等
Material Design Search  搜索组件

# 集成方式

## 添加依赖

1. 在Project的**build.gradle**添加仓库地址

```
allprojects {
	repositories {
		...
		maven { url 'https://www.jitpack.io' }
	}
}
```


```groovy
dependencies {
	  implementation 'com.github.ZLYang110:TitleBar:1.1.0'

}
```
